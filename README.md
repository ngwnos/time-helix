# Time Helix

Interactive fractal coil calendar. All of human history mapped onto a recursive helix — centuries contain decades, decades contain years, years contain days, and so on. Built with Three.js WebGPU.

## Controls

- **Scroll wheel** — zoom in/out between time scales
- **Right-click drag** — pan through time
- **Space** — play/pause
- **Click event labels** — navigate to historical events
- **Level buttons** (left panel) — jump to a specific time scale

## Run

```
bun install
bun run dev
```

Requires a browser with WebGPU support.
